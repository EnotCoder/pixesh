## Переиспользуемые виджеты (src/ui.rs)

### `btn_min_w(ui, label, min_w) -> bool`

Текстовая кнопка с минимальной шириной.

- Ширина: `max(label.len() * CHAR_W * (font_sz / FONT_SZ), min_w)`
- Паддинг: `(14.0, 8.0)`
- Скругление: `4.0`
- Рамка: `Stroke(2.0, BORDER)`
- Состояния: клик=`ACCENT`, ховер=`HOVER`, обычный=`PANEL`

Используется в: Resize dialog ("Apply", "Cancel"), Brush dialog ("OK")

---

### `icon_btn(ui, tex_id, active) -> bool`

Квадратная кнопка с иконкой (текстурой).

- Размер: `Vec2::splat(ROW_H + 16.0)` = 38×38 px
- Скругление: `4.0`
- Рамка: `Stroke(2.0, BORDER)`
- Состояния: активная=`ACCENT`, ховер=`HOVER`, обычный=`PANEL`

Используется в: Toolbar (Brush, Eraser, Fill, Eyedropper, Select, Clear, Mirror H/V)

---

### `separator(ui)`

Вертикальная разделительная линия.

- Высота: `ROW_H + 16.0` = 38 px
- Ширина: `4.0` px
- Рамка: `Stroke(2.0, BORDER)`

Используется в: Toolbar (между логотипом и инструментами, между инструментами и очисткой)

### Обработка инструментов

| Инструмент | События | Поведение |
|------------|---------|-----------|
| Eyedropper | `clicked_by(Primary)` | Берёт цвет с холста |
| Fill | `clicked_by(Primary)` | Flood fill |
| Select | `drag_started/dragged/drag_stopped` | Рисует выделение или перемещает |
| Brush/Eraser | `primary_down/dragged/clicked` | Рисует линию кистью |
| ПКМ ластик | `secondary_down/clicked` | Стирает (TRANSPARENT) |

## Порядок отрисовки (src/app/mod.rs)

```
update():
  1. handle_input(ctx)              — клавиатура, зум, панорама
  2. if show_top_panel: ui_toolbar  — верхняя панель
  3. if show_right_panel: ui_layers — правая панель
  4. ui_canvas                      — центральная область (холст)
  5. ui_dialogs                     — диалоги поверх всего
```
